# \# TDS Builder

# 

# TDS Builder is a node-based pipeline design tool.

# 

# It allows you to visually define:

# 

# \- Project folder structures

# \- Naming conventions

# \- Department workflows

# \- Task dependencies

# \- Global tokens

# 

# \## How It Works

# 

# You create nodes and connect them together.

# 

# Each node:

# 

# \- Receives input values

# \- Processes those values

# \- Outputs new values

# 

# The graph can then be evaluated to generate paths, names, and other pipeline data.

# 

# \## Main Features

# 

# \- Node-based workflow

# \- Nested task graphs

# \- Global tokens

# \- Save and load `.tds` files

# \- Type-safe connections

# \- Property editor
